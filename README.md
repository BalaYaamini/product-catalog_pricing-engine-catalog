# Product catalog pricing & Engine catalog
The Product Catalog & Pricing Engine is a system that manages product information and dynamically calculates pricing based on predefined rules. The product catalog stores details like product name, category, and base price, while the pricing engine applies logic such as discounts, taxes, and special offers to determine the final price.

This project is widely used in e-commerce platforms to ensure accurate, flexible, and real-time pricing. It helps businesses efficiently manage products and optimize pricing strategies while improving the overall user experience.
